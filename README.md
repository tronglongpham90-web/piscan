# Piscan - Pi Network 区块链浏览器

> 基于 https://piscan.io/ 的克隆项目

---

## 🚀 快速开始

### 本地运行
```bash
cd /home/jzy/桌面/piscan/piscan-exact-clone
python3 -m http.server 8080
```

访问: http://localhost:8080

### 部署到 Vercel
```bash
vercel --prod
```
